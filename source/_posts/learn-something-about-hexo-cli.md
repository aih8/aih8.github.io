### 命令工具
```shell
// init
npm install -g hexo-cli
hexo init hexo-blog
cd hexo-blog
npm install

// 多端使用
git clone git@github.com:aih8/aih8.github.io.git --depth=1
cd aih8.github.io
npm i

// localhost server on 4200
npm start

// deploy && git update hexo branch
npm run build
```
